********
關於本書
********

隨著程式語言功能的增強，初學程式設計的困難度不斷的增加。\
Java 在國內許多大專院校，都是基礎程式設計課程的主要教授項目之一。\
初學者使用 Java 作為第一個認識的程式語言，\
可能遭遇許多複雜、抽象的概念與開發工具，\
初學者將會遭遇的困難與挫折實在是可想而知。
